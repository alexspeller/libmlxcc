xcc_content_from_string

(no version information, might be only in CVS)

xcc_content_from_string -- Inserts a new document into the database from a string

Description

resource xcc_content_from_string ( string content_string, string uri [, resource link] )

Creates a new resource for inserting a document into the database from a string. This function behaves exactly like xcc_content_from_file() except it reads from a string instead of a file.

Parameters

content_string

The contnet to be inserted into the database in string form.

uri

The uri that the document will be given once inserted into the database.

link

The connection to the Mark Logic server. If the link identifier is not specified, the last link opened by xcc_connect() is assumed. If no such link is found, an E_WARNING level warning is generated.

Return Values

Returns a statement handle that can be passed into xcc_insert_content()().

Examples

Example 1. xcc_content_from_string() example

<?php
xcc_new_session($user, $password, $host, $database, $port) or die("Counld not connect to server: " . xcc_error());

// create a new document insert handle for the string
$doc = xcc_content_from_string(<book><title>New Book></title></book>, $uri)
    or die("Failed to created insert handle: " . xcc_error());

// inserts the document, returns the number of bytes inserted
$bytes = xcc_insert_content($doc) or die("Failed to insert document: " . xcc_error());

echo "Inserted $bytes bytes from string<br/>";

xcc_close();
?>
