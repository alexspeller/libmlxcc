xcc_fetch_next -- Fetches the next item in a result sequence

Description

array xcc_fetch_next ( resource results )

Fetches the next item in a result sequence.

Parameters

results

The resource returned from xcc_content_from_file() or xcc_content_from_string().

Return Values

Returns a hash of the next item in the result list. This has has two keys: 'type' and 'data'. The type key holds string of the XML Schema type that was returned. The data key holds the data of the item. This data is stored as a string.

Examples

Example 1. xcc_fetch_next() example

<?php
// Find all of the titles in articles and get them out as strings
$results = xcc_submit_query('for $i in /article//title return string($i)') or die("Can't execute query: " . xcc_error());

// Itterate over the the items and print out the 'data' from each
while($item = xcc_fetch_next($results)) {
    echo $item['data'], "<br/>";    // you can get the XML Schema type out with the 'type' key
}
?>
