xcc_content_namespace

(no version information, might be only in CVS)

xcc_content_namespace -- Sets the namespace of the document being inserted

Description

resource xcc_content_namespace ( resource document, integer namespace )

Sets the default xml namespace of the document being inserted if one does not exist on the document.

Parameters

document

The resource returned from xcc_content_from_file() or xcc_content_from_string().

namespace

The default xml namespace to be set on the document if one is not defined on the document.

Return Values

Returns TRUE on success or FALSE on failure.

Examples

Example 1. xcc_content_namespace() example

<?php
// create a new document insert handle for the file
$doc = xcc_content_from_file($file, $uri) or die("Failed to created insert handle: " . xcc_error());

// set the namespace of the document
xcc_content_namespace($doc, "http://www.w3.org/1999/xhtml");

// inserts the document, returns the number of bytes inserted
$bytes = xcc_insert_content($doc) or die("Failed to insert document: " . xcc_error());

xcc_close();
?>
