xcc_close -- Closes the connection to a Mark Logic Server

Description

bool xcc_close ( [resource link] )

xcc_close() closes the connection to the Mark Logic server that's associated with the specified link. If link isn't specified, the last opened link is used.

Parameters

link

The connection to the Mark Logic server. If the link identifier is not specified, the last link opened by xcc_connect() is assumed. If no such link is found, an E_WARNING level warning is generated.

Return Values

Returns TRUE on success or FALSE on failure.

Examples

Example 1. xcc_close() example

<?php
$link = xcc_new_session('xdbc_user', 'xdbc_password', 'localhost', 'Documents', 8002);
if (!$link) {
    die('Could not connect to server: ' . xcc_error());
}
echo "Connected successfully to: ", xcc_server_info($link);
xcc_close($link);
?>
