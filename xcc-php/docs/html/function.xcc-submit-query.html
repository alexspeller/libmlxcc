xcc_submit_query -- Executes a query

Description

resource xcc_submit_query ( string query [, resource link] )

Executes a query an returns the result set. This function differs from xcc_new_adhoc_query() in that it does not allow you set any bind variables. For queries that don't require bind variables this function is handy because it saves you a call to xcc_submit_query().

Parameters

query

The query to be executed.

Note: This query can not take in bind variables.

link

The connection to the Mark Logic server. If the link identifier is not specified, the last link opened by xcc_connect() is assumed. If no such link is found, an E_WARNING level warning is generated.

Return Values

Returns a result resource on success, or FALSE on failure.

Examples

Example 1. xcc_submit_query() example

<?php
// Connect to the database
xcc_new_session($user, $password, $host, $database, $port) or die("Counld not connect to server: " . xcc_error());

// Find all of the titles in articles and get them out as strings
$results = xcc_submit_query('for $i in /article//title return string($i)') or die("Can't execute query: " . xcc_error());

// Itterate over the the items and print out the 'data' from each
while($item = xcc_fetch_next($results)) {
    echo $item['data'], "<br/>";    // you can get the XML Schema type out with the 'type' key
}
?>
