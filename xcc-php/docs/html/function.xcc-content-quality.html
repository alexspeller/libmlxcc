xcc_content_quality

(no version information, might be only in CVS)

xcc_content_quality -- Sets the quality of the document being inserted

Description

resource xcc_content_quality ( resource document, integer quality )

Sets the quality of the document being inserted. Qualities of documents are integers that will be used to weight documents during searches.

Parameters

document

The resource returned from xcc_content_from_file() or xcc_content_from_string().

quality

Any integer, the higher the number the better the score. However, document quality is relative to the other documents returned.

Return Values

Returns TRUE on success or FALSE on failure.

Examples

Example 1. xcc_content_quality() example

<?php
// create a new document insert handle for the file
$doc = xcc_content_from_file($file, $uri) or die("Failed to created insert handle: " . xcc_error());

// set the quality of the document to 100
xcc_content_quality($doc, 100);

// inserts the document, returns the number of bytes inserted
$bytes = xcc_insert_content($doc) or die("Failed to insert document: " . xcc_error());

xcc_close();
?>
