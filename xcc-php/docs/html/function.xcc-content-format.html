xcc_content_format

(no version information, might be only in CVS)

xcc_content_format -- Sets the format of the document being inserted

Description

resource xcc_content_format ( resource document, integer format )

Sets the format of the document being inserted. Format types are: XML, binary, text and none. If type format is set to XML the server will verify that it is well formed. Otherwise the server simply inserts the content into the database. If the type is none the server will decide what the content type is on its own.

By default the server will expect XML as the format type.

Parameters

document

The resource returned from xcc_content_from_file() or xcc_content_from_string().

format

Should be set to one of: XCC_CONTENT_FORMAT_BINARY, XCC_CONTENT_FORMAT_BINARY, XCC_CONTENT_FORMAT_TEXT, XCC_CONTENT_FORMAT_NONE

Return Values

Returns TRUE on success or FALSE on failure.

Examples

Example 1. xcc_content_format() example

<?php
// create a new document insert handle for the file
$doc = xcc_content_from_file($file, $uri) or die("Failed to created insert handle: " . xcc_error());

// set the format of the document to text
xcc_content_format($doc, XCC_CONTENT_FORMAT_TEXT);

// inserts the document, returns the number of bytes inserted
$bytes = xcc_insert_content($doc) or die("Failed to insert document: " . xcc_error());

xcc_close();
?>
