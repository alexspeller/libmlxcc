xcc_error

(no version information, might be only in CVS)

xcc_error -- Returns the error from the previously failed operation

Description

string xcc_error ( [resource link] )

Returns the error that occured during the last function call. If the error was raised from a XQuery statement then the error is returned as a string of XML.

Parameters

link

The connection to the Mark Logic server. If the link identifier is not specified, the last link opened by xcc_connect() is assumed. If no such link is found, an E_WARNING level warning is generated.

Return Values

Returns the error from the last XCC function, or '' (empty string) if no error occurred.

Examples

Example 1. xcc_error() example

<?php
$link = xcc_new_session('bogus_user', 'bogus_password', 'localhost', 'Documents', 8002);
echo xcc_error();
?>
