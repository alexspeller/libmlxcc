(no version information, might be only in CVS)

xcc_content_collections -- Sets the collections that the document belongs to

Description

resource xcc_content_collections ( resource document, array collections )

Sets the collections that the document should belong to.

Parameters

document

The resource returned from xcc_content_from_file() or xcc_content_from_string().

collections

Takes in an array of strings, each entry of the array should be the name of one collection.

Return Values

Returns TRUE on success or FALSE on failure.

Examples

Example 1. xcc_content_collections() example

<?php
// create a new document insert handle for the file
$doc = xcc_content_from_file($file, $uri) or die("Failed to created insert handle: " . xcc_error());

// set the collections the document belongs to
xcc_content_collections($doc, array("Drafts", "Documents"));

// inserts the document, returns the number of bytes inserted
$bytes = xcc_insert_content($doc) or die("Failed to insert document: " . xcc_error());

xcc_close();
?>
