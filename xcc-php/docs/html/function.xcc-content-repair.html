xcc_content_repair -- Attempt to repair the XML file being inserted if needed

Description

resource xcc_content_repair ( resource document, boolean repair )

If the document being inserted into the database is XML and is not well formed the server will attempt to repair the content.

Parameters

document

The resource returned from xcc_content_from_file() or xcc_content_from_string().

repair

When set to true the server will attempt to repair the XML document if needed.

Return Values

Returns TRUE on success or FALSE on failure.

Examples

Example 1. xcc_content_repair() example

<?php
// create a new document insert handle for the file
$doc = xcc_content_from_file($file, $uri) or die("Failed to created insert handle: " . xcc_error());

// try to repair the document if needed
xcc_content_repair($doc, true);

// inserts the document, returns the number of bytes inserted
$bytes = xcc_insert_content($doc) or die("Failed to insert document: " . xcc_error());

xcc_close();
?>
