(no version information, might be only in CVS)

xcc_server_info -- Gets information about the server that you are connected to

Description

resource xcc_server_info ( [resource link] )

Returns a string of information about the server that you are connected to.

Parameters

link

The connection to the Mark Logic server. If the link identifier is not specified, the last link opened by xcc_connect() is assumed. If no such link is found, an E_WARNING level warning is generated.

Return Values

Returns a string with information about the server you are connected to.

Examples

Example 1. xcc_server_info() example

<?php
$link = xcc_new_session('xdbc_user', 'xdbc_password', 'localhost', 'Documents', 8002);
if (!$link) {
    die('Could not connect to server: ' . xcc_error());
}
echo "Connected successfully to: ", xcc_server_info($link);
xcc_close($link);
?>
